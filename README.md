Dice Roller
===========

Roll... Roll... Roll...

## Usage

```
Usage:
  ./roll.sh [n]d[s]

Options:
  n      number of dice(s)
  s      dice side
```

## Examples:
- Roll 3 d20:

```
~$ ./roll.sh 3d20
```

- Roll 1 d2:

```
~$ ./roll.sh d2
```

- Roll 5d2, 4d8 and 1d20:

```
~$ ./roll.sh 5d2 4d8 d20
```
